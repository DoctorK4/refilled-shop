# refilled-shop
이 프로젝트는 콘스탄트 사전과제를 위해 제작된 프로젝트로서, 리필드 샵 레이아웃을 바탕으로 기능들을 구현하였습니다. 

## 배포 주소
[https://refilled-shop.vercel.app/](https://refilled-shop.vercel.app/)

## Stacks
`Next.js`, `Typescript`, `React-Redux`, `SCSS`, `Axios`, `react-select`

## 구현 기능
- lint 및 format 설정
- API 상품 목록 불러오기
- 상품목록 / 장바구니 페이지 라우팅
- 상품별 옵션 선택
- 장바구니 추가
- 장바구니 삭제
- 장바구니 합계 금액 계산
- 반응형 디자인 적용
